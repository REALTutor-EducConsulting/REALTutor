# **<span style="color:#7b2cbf;">REAL Tutorについて</span>**	

REAL Tutorは、既存の塾や家庭教師にない**安心感**と、**外資コンサルティング会社での勤務経験を活かした独自の指導内容**を特徴とする、**新しい学習支援サービス**です。

- 提供サービス：
	- REAL Tutorのサービスは「個別指導」と「デイリーサポート」の2種類から成り立っています。2種のサービスを組み合わせていただくことで最大の効果を発揮できるため、両方のご利用をお勧めしておりますが、どちらかのみのご利用も可能です。
		- 個別指導：「考える力」を「わかりやすく伝授する」独自のマンツーマン指導で生徒様をサポートします。**生徒様の課題が特定できておらず、優先的に取り組む強化・単元のスケジュール作成・参考書選びのアドバイス・モチベーション管理など、総合的なサポートが必要な生徒様に向いています。**
			- 実際のご利用例
				- 中学受験を控えているが、国語がどうしても苦手で、本人のモチベーションも低いためなかなか模試の成績が上がらない。保護者も中学受験経験がないため、どう対処すべきかわからない
				- 大学受験に向け、英語・数学の得点を向上させたい。問題に取り組む際の考え方が定まっていない・スケジュール管理ができていないなど、課題が広範なためまとめてサポートをお願いしたい
				- 中学生になってから勉強についていけず、全教科満遍なくサポートしてほしい
